## Money Transfer API

Money Transfer API
Project Overview
This project is a Money Transfer API that allows users to register, log in, transfer money, and verify transactions. It provides a seamless way to perform transactions, ensuring that all money transfers are verified via email, with verification codes sent to users.

Key features include:

User Registration and Authentication using email and password.
Money Transfer between users, based on phone number.
Transaction Verification via a verification code sent to the user's email.
Password Reset functionality.
Firebase Token Management for notifications on active devices.
